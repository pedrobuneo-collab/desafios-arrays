# Atividade: Estrutura de Vetores em JavaScript

Este repositório contém a resolução de quatro problemas de lógica de programação propostos para fixação de conceitos sobre **Estruturas de Dados (Vetores/Arrays)**. 

### 🚫 Restrição Técnica
Para fins de aprendizado de lógica algorítmica pura, todos os programas foram desenvolvidos **sem o uso de funções ou métodos nativos de arrays do JavaScript** (como `push()`, `pop()`, `indexOf()`, `includes()`, `filter()`, `map()`, etc.). Toda a manipulação de índices e buscas foi feita utilizando estruturas de repetição manuais (`for`).

---

## 🚀 Tecnologias Utilizadas
* **JavaScript (ES6+)**
* **Node.js** (ambiente de execução)
* **Git & GitHub** (versionamento de código)

## 📋 Exercícios Desenvolvidos

### 1. Soma de Números Pares (`exercicio1.js`)
* **Descrição:** Lê 10 números inteiros digitados pelo usuário, armazena-os em um vetor e, ao final, calcula e exibe apenas a soma dos números que são pares.

### 2. Vetor com Números Aleatórios (`exercicio2.js`)
* **Descrição:** Preenche de forma automatizada um vetor de 7 posições com números pseudo-aleatórios gerados pelo computador e exibe-os na tela.

### 3. Contador de Ocorrências (`exercicio3.js`)
* **Descrição:** Solicita 10 números ao usuário, armazena em um vetor e depois pede um número "chave". O programa varre o vetor e diz quantas vezes aquele número específico apareceu.

### 4. Remoção de Valores Duplicados (`exercicio4.js`)
* **Descrição:** Lê 10 números do usuário, armazena-os e cria um segundo vetor contendo os mesmos elementos, porém eliminando qualquer repetição. Exibe o vetor original e o vetor limpo na tela.
